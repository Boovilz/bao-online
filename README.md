# Banlao Academy Online (BAO)

Main Town Alpha V0.1 สร้างด้วย React, Vite และ Phaser 3

## เริ่มใช้งานบน Windows 11

1. ติดตั้ง Node.js รุ่น LTS
2. เปิด Terminal ในโฟลเดอร์โปรเจกต์
3. รันคำสั่ง

```bash
npm install
npm run dev
```

4. เปิด URL ที่ Vite แสดง เช่น `http://localhost:5173`

## ระบบที่มีแล้ว

- เมืองหลักแบบ 2D
- เดินด้วย WASD หรือปุ่มลูกศร
- กล้องติดตามผู้เล่น
- อาคาร 6 แห่ง
- กด E เพื่อเข้าอาคาร
- คลิกอาคารได้โดยตรง
- หน้าต่างรายละเอียดอาคาร
- ฉากสร้างด้วย Phaser Graphics จึงไม่พึ่งภาพเกมที่มีลิขสิทธิ์
